Biografie

Ik ben een enthousiaste en leergierige man uit Nederland. Ik analyseer problemen graag en verbeter systeem, zo nodig met een fikse discussie. Mijn hobbies zijn zowat al het digitale. Verbeteren van design en multimediaal opgegroeid zijn de redenen dat CMD mij op het lijft geschreven lijkt. Het liefste wil ik later systemen die visueel communiceren met de gebruiker vormgeven.

Kennis van HTML en CSS

HTML en CSS zijn mij niet vreemd. Het uitlijnen van video's en blokken vind ik nog steeds erg lastig, maar als dit eenmaal werkt hoef je hier niet veel meer naar te kijken. Momenteel ben ik bezig met een grappig vragenlijstje dat automatisch een verhaaltje genereert voor mijn kinderen. Deze vind je hier.
